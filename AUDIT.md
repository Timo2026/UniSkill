## 二、数据结构

### 统一Skill格式 (skill_schema.py)

```python
Skill:
  - id: 唯一标识
  - name: 技能名称
  - description: 描述
  - skill_type: GENERATOR/ANALYZER/TRANSFORMER/CONNECTOR/UTILITY
  - inputs: 输入定义列表
  - outputs: 输出定义列表
  - metadata: 作者、版本、标签等
  - source: local/clawhub/github
  - keywords: 关键词列表
  - status: INSTALLED/CACHED/AVAILABLE/BROKEN
  - usage_count: 使用次数
  - success_rate: 成功率
```

### 意图对象 (Intent)

```python
Intent:
  - raw_input: 原始输入
  - keywords: 提取的关键词
  - intent_type: 意图类型
  - domain: 领域
  - confidence: 置信度
  - subtasks: 子任务列表
  - constraints: 约束条件
```
